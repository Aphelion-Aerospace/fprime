Detailed Description

This class is an implementation of the MemAllocator base class. It uses the heap as the memory source, but aligns the allocation. The allocation sizes must be a multiple of the alignment, based on the man page for aligned_alloc()

Since it is heap space, the identifier is unused, and memory is never recoverable.
